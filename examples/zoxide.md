# Saltar a una ruta conocida
```bash
z projects
```

# Buscar con varias palabras
```bash
z projects cli tools
```

# Volver al home
```bash
z
```

# Seleccionar interactivamente
```bash
zi
```

# Selección interactiva filtrada
```bash
zi lynx
```

# Regresar al directorio anterior
```bash
z -
```

# Listar rutas registradas
```bash
zoxide query -l
```

# Ver rutas y puntuaciones
```bash
zoxide query -ls
```

# Obtener una ruta sin entrar
```bash
zoxide query lynxwebex
```

# Agregar una ruta manualmente
```bash
zoxide add /ruta/al/proyecto
```

# Eliminar una ruta
```bash
zoxide remove /ruta/antigua
```
